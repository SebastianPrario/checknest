 Gestion de Cheques

## Description

Servidor para la aplicación de Gestión de Cheques  construido con Nest.js y utiliza PostgreSQL como base de datos y TypeORM.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
